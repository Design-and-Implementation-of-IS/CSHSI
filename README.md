# DentalCare – HW2T

## 1. Project Title and Description
DentalCare is a university project focusing on managing a dental clinic’s operations. This part (HW2T) emphasizes the inventory system and XML integration.

## 2. Features
- Inventory tracking: items, categories, types, statuses, and suppliers
- XML data import from "Supir"
- Low-stock and expiration alerts
- Daily inventory snapshot
- DAO pattern for database access

## 3. Technologies Used
- Java 17
- MS Access with UCanAccess JDBC
- XML parsing
- Git + GitHub

## 4. Folder Structure
- src/Entity – domain model classes
- src/Control – business logic and DB access
- accdb – MS Access database file
- diagrams/ – UML diagrams
- xml/ – sample import file

## 5. How to Run
- Requires Java JDK 17+
- Compile using your preferred IDE or command line
- Run via JAR or directly from IDE
- Ensure the DB path and XML files are set correctly
